# 콘텐츠 배포 스킬

## 역할

외부에서 전달받은 콘텐츠를 검토하고 정제하여 One-Pi 홈페이지에 배포합니다.

## 동작 순서

### 1. 콘텐츠 형식 확인

전달받은 콘텐츠가 아래 조건을 충족하는지 확인합니다.

- 제목이 있는가
- 본문 내용이 있는가
- 이미지 등 첨부 파일이 있는 경우 올바른 형식인가 (jpg, png, webp 등)
- 필수 메타 정보(작성일, 카테고리 등)가 포함되어 있는가

형식 오류가 있을 경우 담당자에게 오류 내용을 안내하고 수정을 요청합니다.

### 2. 자동 정제

형식 확인을 통과한 콘텐츠를 자동으로 정제합니다.

- 맞춤법 및 띄어쓰기 교정
- 불필요한 공백, 특수문자 제거
- HTML 태그 이스케이프 처리
- 이미지 최적화 (리사이즈, 압축)

### 3. 담당자 승인

정제된 콘텐츠를 담당자에게 전달하여 최종 승인을 요청합니다.

- 정제 전/후 내용을 비교하여 제공
- 담당자가 내용을 확인하고 승인 또는 반려
- 반려 시 사유를 기록하고 수정 후 재검토

### 4. 배포

담당자 승인이 완료된 콘텐츠를 홈페이지에 반영합니다.

- `index.html` 및 관련 페이지 업데이트
- 변경 사항 커밋 및 배포 실행
- 배포 완료 후 담당자에게 결과 통보
